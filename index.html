О проекте

Веб-приложение для быстрого доступа к админке роутеров. Хранит логины и пароли для различных моделей роутеров и модемов, позволяет быстро копировать учётные данные и открывать настройки роутера.

Инструкции

Использование дефолтных данных:

Выберите роутер из списка, нажмите на карточку для выбора, затем используйте кнопку копирования для логина/пароля или кнопку «Открыть настройки» для перехода в админку.

Добавление своего роутера:

Откройте «Инструменты» → «Мои роутеры», заполните форму (бренд, модель, IP адрес, пары логин/пароль) и нажмите «Сохранить».

Проверка доступности:

После выбора роутера используйте кнопку «Проверить» для проверки доступности роутера через ping.

Резервное копирование:

В разделе «Мои роутеры» найдите секцию «Резервное копирование» для экспорта или импорта настроек.
